# 数独とは、9×9のマスに1から9までの数字を、縦・横の各列と3×3のブロック内で重複しないように埋めていくパズルです。
# あらかじめいくつかのマスに数字がヒントとして入っており、その数字を元にすべてのマスを埋めていきます。
# 数独のルール
# 縦列:1から9までの数字を1つずつ入れる。﻿
# 横列:1から9までの数字を1つずつ入れる。﻿
# ブロック:3×3のブロック内でも、1から9までの数字を1つずつ入れる。﻿
# 初期配置:一部のマスにはあらかじめ数字が埋められています。﻿
# クリア条件:すべてのマスに数字が入り、上記のルールが守られている状態になればクリアです。﻿ 

[数独] ->> [セル] : include

# マスをセルと呼ぶことにします。
# ※セルは、行と列を使って対応付けする必要があります。行、および列は、1～9の数値。
# ※セルには、数字を設定できるようにする必要があります。

[セル] <- [ヒントセル] : subtype
[セル] <- [入力セル] : subtype

# 入力セルには、候補の数字を設定することができる。
