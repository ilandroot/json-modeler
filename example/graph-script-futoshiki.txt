# model: futoshiki 
# 不等式パズル（一般的に不等号パズルやFutoshikiと呼ばれる）は、マス目に数字を埋めていく論理パズルの一種です。
# 数独（ナンプレ）に似ていますが、数字の大小関係を示す「不等号」がヒントとして与えられるのが特徴です。 
# 基本ルール
# 数字の配置: N x N のマス目に、1から N までの数字を1つずつ入れます（例：5×5の枠なら1〜5を使用）。
# 重複禁止: 各行および各列に、同じ数字が重複して入ってはいけません。
# 不等号の遵守: マス目とマス目の間にある不等号（< や >）の条件をすべて満たすように数字を選びます。 

# 関係式
[不等式パズル] ->> [セル] : include
# マス目を、セルと呼ぶ

[不等式パズル] ->> [不等号] : include
[セル] <- [不等号] : ref (lower)
[セル] <- [不等号] : ref (upper)

[セル] <- [ヒントセル] : subtype
[セル] <- [入力セル] : subtype
# セルには、初めから数字が入っているヒントセルと、数字を入力する入力セルがある。
