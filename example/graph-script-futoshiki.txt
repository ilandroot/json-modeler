# 「パズル 不等式」とは、数独（ナンプレ）のルールに「不等号」（＞や＜など）の要素を加えた数字パズルのことで
# マス目に数字を配置する際に、隣接するマスの大小関係が不等号の示す通りになるように矛盾なく埋めていく論理パズルです。
# 数独のように各列・各行で数字が重複しないルールに加えて、「左のマスは右のマスより大きい（小さい）」といった制約を満たすのが特徴で
# 脳トレや論理的思考力を鍛えるのに人気があります。 
# 詳しいルールと特徴
# 基本ルール（数独系）:
# タテ列、ヨコ列に指定された数字（例：1〜5または1〜9）が重複しないように配置します。
# 数独のルールに似ていますが、計算は不要で、論理的な配置が求められます。
# 不等号ルール（パズル要素）:
# マスとマスの間に配置された不等号（＞、＜）が指し示す通りに、数字の大小関係を成立させます。
# 例：「5マス」のパズルで、「3」と「1」の間に「＞」があれば、「3」が「1」の左側に来る、といった具合です。
# 別名: 「不等式ナンプレ」「不等号ナンプレ」「フツーシキ」などとも呼ばれます。
# 難易度と形式: 5x5から9x9マスまであり、問題作成機能で無限に楽しめるアプリも存在します。 
[不等式] ->> [セル] : include
[セル] <- [ヒントセル] : subtype
[セル] <- [入力セル] : subtype
# ここまでは、「数独」と同じ
# 不等式（パズル）には、サイズを変更して問題を作成できる。想定として、サイズは、4～9であり、それ以上でも構わないが、10をすぎると、A,B,C,...のように表示する。
[不等式] ->> [不等号] : include
[セル] <- [不等号] : ref (lower)
[セル] <- [不等号] : ref (upper)
# 不等号の数および、設定位置は、問題によって変わる。これらは、制約として機能する。
