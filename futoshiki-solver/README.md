# Futoshiki Solver (不等式パズル ソルバー)

このリポジトリは、4×4 の Futoshiki（不等式パズル）を  
**制約伝播（Constraint Propagation）** によって解く Node.js プログラムです。
問題データは、data.txtに定義します。

行・列の重複禁止制約と、不等号制約（<, >）を組み合わせ、  
候補削減を反復適用することで盤面を自動的に解きます。

---

## ✨ 特徴

- **弱い不等号制約（upperMax / lowerMin）** による安全な候補削減  
- **行列制約** による候補の除外  
- **単一候補確定（singleton fix）** による自動確定  
- **収束まで反復** することで自然に解が導かれる  
- **無限ループ防止** のため、進展が止まったら終了  
- **解けた場合は盤面をフラットに表示**

---

## 🧩 プログラムの使い方

### 1. パズルデータを用意する
`data.txt` に、以下の形式でパズルを記述します。

- 1 行目：盤面サイズ（例：4）
- 続く N 行：初期盤面（0 は未設定セル）
- 空行のあと：不等号制約（(r,c) < (r,c) の形式）


---

### 2. ソルバーを実行する

Node.js がインストールされている環境で、次のコマンドを実行します。

```bash
node solver.js
